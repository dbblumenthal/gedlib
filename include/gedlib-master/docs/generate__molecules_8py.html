Detailed Description

Python script for generating synthetic molecules.

The synthetic molecules generated by this script were used for the experiments in the following paper:

  • D. B. Blumenthal, S. Bougleux, N. Boria, J. Gamper, L. Brun: “Comparing heuristics for graph edit distance computation”, Accepted for publication in VLDB J.

Usage:

$ python generate molecules.py
Warning
Running this script overrides the molecules distributed with GEDLIB that were used for the experiments in the VLDB J. paper.
